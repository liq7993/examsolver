设置 · LLM 服务商

选择一个云端大模型来生成「学生解释」。确定性解题（求导、矩阵、受力平衡等）由本地 sympy 内核完成，无需 API key、永远免费。

⚠ 使用第三方 API 前请注意

  • 题目内容会发送到所选服务商，请勿提交隐私或涉密信息。
  • 请在服务商后台设置消费上限，避免意外扣费。
  • API key 等同密码：不要泄露、不要截图、不要提交到 Git。
  • key 仅保存在本机服务器，界面只显示末四位。

欢迎使用 ExamSolver

三步开始：

  1. 直接出题：求导、矩阵乘法、受力平衡等由本地 sympy 内核求解，无需联网、无需 API key。
  2. 想要「学生解释」？ 在「设置」里选择一个云端大模型（如 DeepSeek / OpenAI / Moonshot），粘贴它的 API key。
  3. 如何获取 key：前往服务商官网注册 → 控制台「API Keys」创建 → 复制以 sk- 开头的字符串。建议先充小额并设置消费上限。

⚠ 安全提示

  • API key 等同密码：不要泄露、截图或提交到 Git。
  • 题目会发送给第三方服务商，请勿提交隐私／涉密内容。
  • 务必在服务商后台设置消费上限。
